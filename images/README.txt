INSTRUCCIONES PARA REEMPLAZAR LAS IMÁGENES
===========================================

Reemplaza cada archivo con tus propias fotos de cafetería.
Mantén los mismos nombres de archivo para que funcionen automáticamente.

LISTA DE IMÁGENES NECESARIAS:
------------------------------

SECCIÓN HERO (Principal):
- hero-cafeteria.jpg (600x400px recomendado)
  → Foto principal de tu cafetería, exterior o interior acogedor

SECCIÓN NOSOTROS:
- about-equipo.jpg (500x300px recomendado)
  → Foto del equipo, barista trabajando, o ambiente del local

MENÚ - CAFÉS:
- cafe-espresso.jpg (300x200px)
  → Foto de un espresso
- cafe-cappuccino.jpg (300x200px)
  → Foto de un cappuccino con espuma
- cafe-latte.jpg (300x200px)
  → Foto de un latte

MENÚ - PASTELES:
- pastel-manzana.jpg (300x200px)
  → Foto de tarta de manzana
- pastel-cheesecake.jpg (300x200px)
  → Foto de cheesecake
- pastel-brownies.jpg (300x200px)
  → Foto de brownies

MENÚ - BEBIDAS:
- bebida-jugo.jpg (300x200px)
  → Foto de jugo de naranja
- bebida-te.jpg (300x200px)
  → Foto de té helado
- bebida-limonada.jpg (300x200px)
  → Foto de limonada

GALERÍA (Carrusel):
- galeria-interior.jpg (800x400px)
  → Interior de la cafetería
- galeria-cafe.jpg (800x400px)
  → Productos o preparación de café
- galeria-clientes.jpg (800x400px)
  → Clientes disfrutando en el local

FORMATOS ACEPTADOS:
- .jpg, .jpeg, .png, .webp

CONSEJOS:
- Usa imágenes de buena calidad
- Mantén los tamaños recomendados para mejor rendimiento
- Puedes usar fotos de stock gratuitas de:
  * Unsplash.com
  * Pexels.com
  * Pixabay.com
